# User Management System

## Overview
Simple PHP/MySQL web app to:
- Add users (name + age)
- List users in a table
- Toggle user status (0 ↔️ 1)
- Delete users

## Requirements
- XAMPP (Apache + MySQL + PHP)
- phpMyAdmin
- Database name: info
- Table name: users
